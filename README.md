# 🎮 Pokedaily

매일 00:00(KST)에 갱신되는 새로운 체육관 관장에게 도전하는 1대1 턴제 포켓몬 배틀 게임입니다.

## 🔗 플레이 링크
**[https://pokedaily-tau.vercel.app](https://pokedaily-tau.vercel.app)**

## 🚀 주요 특징
- **매일 새로운 챌린지**: 매일 자정에 모든 유저에게 동일한 체육관 관장과 렌탈 포켓몬 풀이 제공됩니다.
- **전략적 팀 구성**: 주어진 6마리의 렌탈 포켓몬 중 승리할 수 있는 최적의 3마리를 선택하세요.
- **클래식 배틀 시스템**: 타입 상성, 자속 보정(STAB), 스탯 기반 턴제 로직이 구현된 1세대 스타일의 배틀을 즐기세요.
- **결과 공유**: 배틀 결과를 클립보드에 복사하여 친구들에게 자랑할 수 있습니다.

## 🛠 기술 스택
- **Frontend**: React (Vite)
- **Styling**: CSS Modules / Responsive Design
- **Data**: PokeAPI 기반 커스텀 데이터셋
- **Deployment**: Vercel
